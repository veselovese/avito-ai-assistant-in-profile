## **Запускаем проект**
1. Внесены изменения в файл back\`а, так как серверная часть не позволяла совершать полноценную разработку на клиенте - подробнее в Дневнике разработки. Перед запуском проекта необходимо внести это изменение в файле server.ts.
```
fastify.get<ItemsGetRequest>('/items', request => {
    ...

        .map(item => ({
                id: item.id, // вот эту строчку добавил
                category: item.category,
                title: item.title,
                price: item.price,
                needsRevision: doesItemNeedRevision(item),

2. Клонируйте проект `git clone https://github.com/veselovese/avito-ai-assistant-in-profile.git`. Перейдите в его директорию `cd avito-ai-assistant-in-profile`. Установите все зависимости `npm i`.
3. В проекте реализована интеграция с Ollama. Установите Ollama, загрузите модель и запустите.
4. Создайте файл `.env` и скопируйте в него содержимое файла `.env.example`.
5. В переменную `VITE_OLLAMA_URL` вставьте адрес, на котором работает Ollama - `http://localhost:11434`. Переменную `VITE_API_BASE_URL` оставьте пустой - проект локально работает через vite.proxy, который описан в файле vite.config.ts. Причины такого решения описаны в Дневнике разработки.
6. Проверьте, что серверная часть работает, и запустите клиентскую `npm run dev`.

Есть два варианта: я вижу комментарий в коде сервера "Настройка CORS" и могу уснановить нужные header\`ы там или могу описать proxy на стороне vite. На этапе разработки буду использовать proxy, свое решение на back`е прилагаю.
```
fastify.use((_, reply, next) => {
  reply.setHeader('Access-Control-Allow-Origin', '*');
  reply.setHeader('Access-Control-Allow-Methods', 'GET,POST,PUT,OPTIONS')
  reply.setHeader('Access-Control-Allow-Headers', 'Content-Type')

  if (_.method === 'OPTIONS') {
    reply.statusCode = 204
    reply.end()
    return
  }
  next();
});
```

const cleanParams = (params: Record<string, any>) => {
        return Object.fromEntries(
            Object.entries(params).filter(([_, value]) => {
                return value !== '' && value !== null && value !== undefined;
            })
        );
    };  
```
